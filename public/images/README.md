# Images Directory

## BHHS Logo

Place your Berkshire Hathaway HomeServices logo file here:
- File name: `bhhs-logo.png` (or `.jpg`, `.svg`)
- Recommended size: 240x80px or similar aspect ratio
- Format: PNG with transparent background preferred

The logo will automatically appear in the footer once the file is added and the path is configured in `app/config/business.ts`.
